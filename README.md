# The Rise Group Web Page

## Description

A promotional webpage for "The Rise Group," a fitness program designed to help women achieve physical and mental strength without burnout. The page highlights a four-week structured workout challenge, nutrition guidance, accountability check-ins, private community support, and daily mindset tips.

### Features

- **Modern Design**: Vibrant hot pink and charcoal styling with white accents.
- **Engaging Content**: Direct call-to-action with a visually appealing layout.
- **Responsive Layout**: Adapts seamlessly across devices.
- **Promotional Section**: Program-specific details and waitlist invitation.

## Deployment
